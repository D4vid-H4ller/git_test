Command Line Interface

commands

pwd - print working directory

ls - list - -F flag/swithch/option --classify

cd - change directory - . current dir - .. parent dir - ~/ home dir - / root

mkdir - make directory - use flag -p to create the parent dirs if needed

touch [file] - creates a file

man [command] - opens the manual

cp - copy

mv - move - second argument is the new name and/or new location

rm - remove - it deletes w/o asking, be careful - add flag -r to delete directories

* - wildcard for 0+ characters

? - wildcard for exactly 1 character

Git and Github

commands

Locate the repository and use the commands:

git status

git log

git add [file] # adds to the staging area

git commit -m "Whatevermessage"

git push origin main

HTML

headings

Headings are ranked from h1 to h6, there are no h7+, so don't try to use them.

There are no p2+ tags either.

tags

strong (important/bold)

em (emphasis/italic)

lists

ul - unordered list

li - list item element

ol - ordered list

links

parts of a URL

anchor element "a"

attributes

href - hypertext reference, must include the "scheme" as in "https://

target="_blank" - opens link in a new tab

rel="noreferrer" - prevents info leakage from source to target pages

image (void) element "img"

src - source

alt - alternative text to display/read if img not available, it should be a description of the img (also used to trick SEO)
