HTML

headings

Headings are ranked from h1 to h6, there are no h7+, so don't try to use them.

There are no p2+ tags either.

tags

strong (important/bold)

em (emphasis/italic)

lists

ul - unordered list

li - list item element

ol - ordered list

links

parts of a URL

anchor element "a"

attributes

href - hypertext reference, must include the "scheme" as in "https://

target="_blank" - opens link in a new tab

rel="noreferrer" - prevents info leakage from source to target pages

image (void) element "img"

src - source

alt - alternative text to display/read if img not available, it should be a description of the img (also used to trick SEO)

JPGs

JPG images are good for photos

GIFs

PNGs

SVGs
