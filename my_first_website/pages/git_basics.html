Git and Github

commands

Locate the repository and use the commands:

git status

git log

git add [file] # adds to the staging area

git commit -m "Whatevermessage"

git push origin main

If the reader wonders what the change was, they can simply take a look i.e. use git show or git diff or git log -p.

git log --oneline, which prints out just the subject line

git shortlog, which groups commits by user, again showing just the subject line for concision
